# テスターの方へ：フィードバックの送り方

WillMeterのユーザーテストにご協力いただきありがとうございます。気づいたことは、些細なことでもぜひ教えてください。

## 方法1: TestFlightアプリから送る（一番手軽）

1. WillMeterを操作していて気になる場面が出てきたら、その場でスクリーンショットを撮る
2. 通知（またはTestFlightアプリ）から「フィードバックを送信」を選ぶ
3. スクリーンショットに直接マーカーで書き込みながら、コメントを添えて送信できる

不具合や「あれ？」と思った瞬間をそのまま伝えられるので、まずはこの方法をおすすめします。

## 方法2: GitHub Issueから送る（じっくり書きたいとき）

再現手順を整理して書きたい場合や、機能の要望などは、GitHub Issueのテンプレートからも送れます。

1. リポジトリの [Issues](../../issues/new/choose) を開く
2. 「ユーザーテストフィードバック」テンプレートを選ぶ
3. フォームに沿って入力して送信する

GitHubアカウントが無い場合や使い方が分からない場合は、方法1（TestFlight）で問題ありません。

## どんな内容でも歓迎です

- 操作していて分かりにくかった・迷った箇所
- 期待した動きと違った箇所（不具合）
- こんな機能があったら良い、という要望
- 良かった点・気に入った点（これも重要な情報です）

## プライバシーについて

フィードバックに含まれる内容（スクリーンショット等）は、アプリの改善のみに使用します。個人が特定される情報は書かないようお願いします。
